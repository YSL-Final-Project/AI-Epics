# AI Code Era — 项目展示说明书

> 2018–2025 AI 编程革命全景交互网站

**线上地址：** https://finalwebdesign-alpha.vercel.app

---

## 项目定位

这是一个**叙事驱动的数据可视化网站**，讲述 AI 如何在 2018–2025 年间彻底改变了软件开发行业。

网站融合了沉浸式滚动叙事、交互式数据图表、游戏化互动体验和电影级视觉设计，将枯燥的数据转化为一场身临其境的故事之旅。

## 全站通用特性

### 中英双语切换
- 右上角一键切换中文/English
- 覆盖所有页面、所有文本、所有交互提示
- 语言偏好自动保存，刷新不丢失

### 深色/浅色主题
- 导航栏滑动开关，带日月图标动画
- 全站配色无缝切换

### 自定义光标
- 鼠标跟随的外圈（弹簧物理）+ 内点（即时响应）
- 悬停到可交互元素时自动放大变色

### 页面滚动进度条
- 顶部彩色细线，弹簧动画跟随滚动

### 欢迎弹窗
- 首次访问 1.8 秒后自动弹出
- 毛玻璃卡片 + 渐变顶部边框
- 6 秒倒计时自动关闭

### 代码窥视窗口（CodePeek）
- 很多特效旁边有 `</> 文件名` 的小按钮
- 点击后从中心展开一个**可拖拽的代码窗口**
- 彩虹旋转边框 + Dracula 主题语法高亮
- 手绘风格的小手来回摆动提示"点击展开"
- 关闭时窗口向中心收缩卷回
- 按 ESC 也可关闭

### 页面懒加载
- 所有页面组件按需加载，首屏极速

## 首页（`/`）

首页分为 4 个章节，用右侧的圆点导航串联：

### 章节一：Hero 主视觉

**演示要点：**
- 全屏黑色背景 + **Matrix 代码雨**（Canvas 动画，绿色字符如雨滴般下落）
- 标题 **逐字 3D 翻转出场**，每个字带旋转透视效果
- 两个 **跟随鼠标的模糊光球**，移动鼠标可以看到它们以不同速度跟随
- **终端模拟器**：
  - 模拟 macOS 终端窗口（红黄绿三个点）
  - 逐字打字动画，速度各行不同
  - CRT 扫描线覆盖效果
  - 打字完成后有 READY 标识
  - **隐藏彩蛋：停留 25 秒不动，终端会自动打出搞笑的程序员语录**

### 章节二：Apple 风格粘性滚动叙事

**演示要点：**
- 滚动时画面不滚走，而是**三个场景交叉淡入淡出**
- 场景一：巨大的"代码"二字 + 代码片段
- 场景二：巨大的"46%" + SVG 进度环
- 场景三："边界正在消失" + 轨道粒子动画
- 左侧有细线进度指示器
- 右侧三个圆点显示当前场景

### 章节三：关键数据仪表盘

**演示要点：**
- 4 张数据卡片：40% AI 代码占比、55% SO 流量下降、1.5M+ Copilot 用户、92% 开发者使用 AI
- 数字滚动计数动画
- 卡片带渐变色条和发光效果
- 弹簧物理的交错出场动画

### 章节四：探索维度

**演示要点：**
- 5 张预览卡片，链接到各子页面
- 鼠标悬停时**3D 倾斜效果**（根据光标位置计算透视角度）
- 卡片角落有装饰性十字线条纹

## 故事页（`/story`）

**这是整个网站的核心体验**——独立的全屏电影级叙事页面，没有导航栏和页脚。

### 全局背景效果

**演示要点：**
- Canvas 粒子**星空**背景（250 颗星，带视差深度）
- **氛围光** 大气照明效果
- **胶片颗粒感**噪点覆盖层

### 叙事结构（从上往下滚动）

#### 序曲（Overture）
- 代码打字动画：`function future() { return human + AI; }`
- "The AI Code Era" 标题从透明到黑体的**字重渐变动画**
- 标题缩放随滚动变化

#### 桌面进化
- 三个 OS 窗口并排：经典 Mac、Windows、Linux
- 代码块从经典代码过渡到 AI 时代代码
- 逐行揭示动画

#### 叙事过渡（3 处）
- 大字文本渐入："在那之前，一切都还很简单"
- **速度缎带**：文字根据滚动速度产生倾斜和位移
- 第二处使用**故障文字效果**（字符随机替换 + 青/红色差偏移）

#### 第一章：暗流涌动（2018–2021）
- **水平滚动时间线**：垂直滚动驱动水平移动
- GPT-1 → GPT-2 → GPT-3 → Copilot → ChatGPT
- 每个节点带脉冲圆点 + 年份 + 描述
- 之后切换到 **SO 流量上升曲线**（SVG 路径动画 + 发光效果）
- 计数器从 1120M 滚动到 1700M

#### 第二章：大爆炸（2022–2024）— 7 个子场景
1. **那一天**：巨大的 "2022.11.30" 数字，缩放 + 模糊入场
2. **SO 流量暴跌**：55% 大数字 + 标注
3. **交叉图表**：AI 采用率上升线 vs SO 流量下降线交叉
4. **40% 里程碑**：GitHub AI 代码占比大数字
5. **行业渗透率**：7 个行业的水平条形图（滚动驱动增长动画）
6. **关键事件级联**：6 个事件沿时间轴依次滑入
7. **章节结语**："700 天。从玩具到基础设施。"

#### AI vs 人类张力（DualWaveText）
- 左列（AI）：速度、规模、自动化、无限、压缩
- 右列（人类）：判断、意义、创造力、直觉、情感
- 两列文字以**正弦波反向摆动**，形成推拉张力

#### 第三章：新物种（2025+）
- **薪资鸿沟**可视化
- **开发者岗位**条形图（2018–2025），2023 后变红色
- 计数器显示 -29% 跌幅
- **IDE 格局**：VS Code / Cursor / JetBrains / Vim 市场份额条形图
- Cursor 从 0% 增长到 18% 的动画

#### 分支时刻（交互选择）

**演示要点：**
- 用户选择：**黎明（乐观者）** or **暮色（现实者）**
- 两个选项带不同色调（青色 vs 红色）
- 选择后展示对应视角的深度论述
- 可以点击"选择另一个视角"切换

#### 尾声

**演示要点：**
- "AI 没有取代开发者。它重新定义了「开发者」的含义。"
- "开发者"三个字带**故障闪烁效果**
- 两个链接按钮：探索完整数据 / 浏览时间线
- 底部极淡的项目署名

## 无障碍支持

- 减弱动画模式：检测系统设置，所有动画组件提供静态回退
- ARIA 标签：按钮、开关、实时区域均有语义化标注
- 键盘导航：ESC 关闭弹窗，标准 Tab 导航
- 色彩对比度：深色/浅色主题均经过可读性验证

---

## 性能优化

- 页面级代码分割（React.lazy 按需加载）
- Canvas 渲染代码雨和星空（requestAnimationFrame 60fps）
- 滚动事件使用 `passive: true`
- 搜索输入 300ms 防抖
- CodePeek 使用 Portal 渲染避免布局抖动
- 数据密集组件使用 `React.useMemo`
- 视口动画 `once: true` 避免重复触发
